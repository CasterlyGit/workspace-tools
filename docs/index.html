04

Philosophy

Solo dev, AI augmented. The harness is for one person who wants the throughput of a team — no daemons, no web app, no permissions model. Markdown files are the API. Skills run visibly in the terminal. Every state can be rebuilt from disk on next invocation.

files are the api

Every interaction crosses a markdown or JSON file boundary. Kill every process and the system rebuilds itself.

visible work

Skills stream in the user's terminal. Read along, interrupt, course-correct. No opaque background pipeline.

fast default

Single-call skills by default. Multi-stage pipelines are opt-in, never silent upgrades.

stateless workers

Each skill invocation re-derives context from disk. Amend cycles are cheap and resumable.
